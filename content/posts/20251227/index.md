高投票率的老年人使用多數人暴力拖累社保，絕望的冰河世代要面對工資還沒有漲上去就要縮水，政府以公司這個中介瘋狂抽取中產階層，年輕人低薪加民粹。日本的未來看起來並不樂觀。

首先聲明我完全無意否認稅跟社保的重要性。到了年末算了一下稅金又重新學了點稅務相關的知識，想以一個打工人的身份來總結一下日本複雜的稅制以及對打工人來說有限的節稅方法。

先擺上數字比較容易理解。一個普通上班族A，拿著略高於平均水平的年收入500萬yen，無撫養，沒有車，工資的50%用於租房以外的消費。

## 舉個例子

那麽A在2025年這個要交的稅是 **36.02%**
這還是在實行的減稅並且有撫養的情況下，如果沒有還會高處很多。

這裡的**36.02%**是怎麼算出來的呢，拆開看看。
從社会保険（年金跟保險跟健康保險）開始算起比較容易理解，因為社会保険是用收入來計算的，稅是用所得（從收入扣掉必要的經費）來計算的。

### 厚生年金

- [稅率是18.30%](https://www.nenkin.go.jp/service/kounen/hokenryo/ryogaku/ryogakuhyo/20200825.html)
- 月收入是416,666屬於410,000那一檔
- 按照自己出一半，公司出一半的原則那麼一年的厚生年金是 410,000 * 0.183 * 0.5 * 12 = **450,180**

### 健康保險

- 根據協會稅率會不同，以[［ITS］関東ITソフトウェア健康保険組合](https://www.its-kenpo.or.jp/hoken/jimu/hokenryou/jigyou.html)為例，稅率是9.50%
- 按照自己出一半，公司出一半的原則: 5,000,000 * 0.095 * 0.5 = **237,500**

### 所得税部分

- [給与所得控除](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1410.htm): 5,000,000 * 0.2 + 440,000 = 1,440,000
  - 可以理解為打工人的經費，只不過這個經費是被釘死的
- [基礎控除](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/1199.htm): 680,000
  - 需要通過「合計所得金額」來算出。所得是 5,000,000 - 1,440,000 = 3,560,000
- 社会保険控除: 450,180 + 237,500 = 687,680
- 所得税 
  - 課稅部分: 5,000,000 - 1,440,000 - 680,000 - 687,680 = 2,192,320
  - [稅率](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/2260.htm) 20%外加97,500的控除
  - 所得税 2,192,320 * 0.1 - 97,500 = **121,732**

### 復興稅部分

- 所得税的1.2%: 121,732 * 0.021 = **2,556**

### 住民税部分

- 基礎控除縮水為: 430,000
- 其餘不變
- 住民税
  - 課稅部分: 5,000,000 - 1,440,000 - 430,000 - 687,680 = 2,442,320
  - 稅率: 10%上下
  - 住民税: 2,442,320 * 0.1 = **244,232**

上面四項全部扣完，A還剩下5,000,000 - 450,180 - 237,500 - 121,732 - 2,556 - 244,232 = 3,943,800
還需要計算消費稅

### 消費稅部分

- 因為房租不用付消費稅，假設A工資的50%用於租房以外的消費，那A花出去的錢裡面，消費稅佔  3,943,800 * 0.5 - (3,943,800 * 0.5) / 1.1 = **179,264**

綜上，A的稅率是 (450,180 + 237,500 + 2,556 + 244,232 + 179,264) / 5,000,000 = 22.27%

不，還需要把公司幫A付的那部分社會保險也算進去，因為這本身也是用工成本的一部分
(450,180 + 237,500 + 2,556 + 244,232 + 179,264 + 450,180 + 237,500) / 5,000,000 = **36.02%**

非常嚇人，一個禮拜工作五天，有幾乎接近兩天是為國家打工。

政府故意把稅相關的制度弄複雜，然後通過公司這個中介直接從打工人手中徵稅，降低打工人對稅的直接感知。（有點陰謀論，但應該差的不多）

## 稅率降低的可能性

未來稅率有可能降低麼？

- 厚生年金，健康保險: 只要老齡化還在，制度不突然崩塌，這兩項不可能減少
- 消費稅: 用於堵上厚生年金，健康保險的窟窿，同理難以減少
- 所得税: [国民民主党](https://new-kokumin.jp/)正在努力的方向，但跟收入超過665萬yen的所謂「高收入」人群基本沒關。即便能減少，也不是房間裡的大象。而且国民民主党已經開始有點慶祝勝利的樣子
- 復興稅: 即將被置換為防衛稅，不太可能減少
- 住民税: 直接觸及到地方的利益，難以減少

很絕望

## 如何節稅

一般來說有三種方法
1. 做打工人的同時做副業
2. 不做打工人，做自由職業
3. 自建法人，用法人付最低限度的社會保險，同時做自由職業來維持生計

副業或者說自由職業的好處在於

- 經費可以根據情況自己決定，詳細在這裡就不講了
- 滿足記帳條件可享受 650,000yen 的青色申告控除
  - 需要注意如果被分類為雑所得就無法享受，[判斷標準](https://www.town.otaki.chiba.jp/material/files/group/11/20230517-100254.pdf)有點複雜
- 每週20小時以內的副業不用繳納社保

如果在往前走一步，不做打工人，做自由職業會怎麼樣。年金部分變為國民年金，每年固定 210,120yen，跟厚生年金比便宜太多。但會帶來另外一個問題，保險變為國民健康保險，根據收入費用會很高。用網上隨便搜的[国民健康保険料シミュレーション](https://www.mmea.biz/simulation/kokuho_calculation/)以東京都中野区來計算，自由職業 3,000,000 所得（不是收入）需要繳納 339,000 每年，比從公司加入的健康保険組合高多了。

同時削減稅和社保的方法其實還是有的，自建法人後用法人支付給自己最低限度的工資，這樣需要支付的社保就能被壓縮到最低。再用自由職業的部分來支付其他各種稅金。缺點是操作過於繁瑣，而且要求法人跟個人做的工作完全不相關。因此做打工人的同時做副業看起來是比較現實的方法。

繼續用A的例子來看

A發現稅高的離譜看了我的上面的文章開始做副業，

- 一年下來A的副業收入是 **1,500,000**
- 假設其中經費是 750,000
- 享受 650,000 的青色申告控除
- 厚生年金，健康保險不會增加
- 新增的課稅對象: 1,500,000 - 750,000 - 650,000 = 100,000
- 需多付的所得稅: 100,000 * 0.1 = 10,000
- 需多付的住民稅: 100,000 * 0.1 = 10,000

A的稅率變為

(450,180 + 237,500 + 2,556 + 244,232 + 179,264 + 450,180 + 237,500 + 10,000 + 10,000) / (5,000,000 + 1,500,000) = **28.02%**

從 **36.02%** 降低到 **28.02%**，可見無需付社保的情況下獲得收入的威力

可見現行制度下做副業是打工人節稅的近乎唯一的方法，但需要犧牲一部分休息時間，不是所有人都有做副業的體力和時間。
